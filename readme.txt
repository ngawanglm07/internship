# JOB seeker app

# A intern project to help recruiters get the resume from job seekers

## Technologies Used

* node.js 
* express
* ejs
* mongoDB

packages like google o-auth security to signup using google and level 3-4 secuirty using packages like passport , passport local mongoose and session .

## Features and Functionality with use :-

*  Users can login or signup using google o-auth secuirty where their login will be stored on mongodb atlas 
*  users can send their resume and submit them online where the data will be forwarded to recruiters
